# Mavis PRD

## 1. 제품 개요

- 프로젝트명: Mavis
- 목적: 스타트업 아이디어를 AI로 빠르게 검증하고, 커뮤니티 피드백과 팀 빌딩, 초기 자금 로드맵까지 한 곳에서 제공하는 플랫폼
- 제품 포지션: 아이디어 검증 + 커뮤니티 + 실행 지원(팀/자금) 통합 서비스
- 핵심 가치: 빠른 검증, 집단지성 피드백, 실행을 위한 구체화(팀/예산)

## 2. 문제 정의

- 창업 아이디어는 초기 단계에서 객관적 검증이 어렵다.
- 커뮤니티 피드백과 신뢰성 있는 비교 지표가 부족하다.
- 팀 빌딩과 초기 자금 계획이 분리되어 있어 실행까지 이어지기 어렵다.

## 3. 목표와 성공 기준

### 비즈니스/제품 목표

- 아이디어 등록부터 실행(팀/자금)까지의 전환 경로를 제공
- 커뮤니티 참여(업보트/댓글)를 활성화

### 성공 지표(예시)

- 아이디어 등록 수, AI 분석 완료율
- 업보트/댓글 수, 피드 참여율
- 팀 참가 신청 수
- 로드맵 작성 및 공유/내보내기 사용률

## 4. 타겟 사용자

- 1차 사용자: 예비 창업자, 아이디어 보유자
- 2차 사용자: 프로젝트 참여를 원하는 개발자/디자이너/PM 등 팀원
- 커뮤니티 참여자: 다른 아이디어를 평가하고 피드백하는 사용자

## 5. 핵심 사용자 시나리오

1. 회원가입/로그인
2. 아이디어 등록 (문제, 타겟, 해결 방안, 차별성 입력)
3. AI 분석 결과 확인 (종합 점수 + 항목별 점수 + 피드백)
4. 피드에서 공개 및 업보트/댓글로 검증
5. 팀 현황 확인 및 참가 신청
6. 초기 자금 로드맵 작성/공유

## 6. 주요 기능 (MVP 기준)

### 6.1 아이디어 등록 및 AI 분석

- 입력 폼 기반 아이디어 등록
- AI가 시장성/혁신성/실현가능성 점수 및 피드백 제공

### 6.2 아이디어 상세 페이지

- 아이디어 상세 정보(문제/타겟/해결/차별성)
- 업보트/다운보트, 댓글
- AI 분석 결과 요약 및 항목별 점수

### 6.3 커뮤니티 피드

- 아이디어 목록 노출
- 정렬(인기순, 점수순) 및 카테고리 필터

### 6.4 팀 빌딩

- 팀 현황 및 모집 중인 역할 표시
- 팀 참가 신청 액션

### 6.5 자금 로드맵

- 초기 비용 템플릿 기반 테이블
- 월/분기별 계획 및 예산 차트
- 공유/내보내기 액션

### 6.6 마이페이지

- 프로필 조회/수정
- 내 아이디어/프로젝트/활동 기록
- 알림 및 보안(비밀번호 변경/계정 삭제/로그아웃)

## 7. 정보 구조 (IA)

- 홈
- 아이디어 등록
- 아이디어 상세
- 피드
- 로드맵
- 마이페이지
- 로그인/회원가입

## 8. 기술 및 시스템 개요

- 프레임워크: Next.js (App Router), React, TypeScript
- UI: Tailwind CSS + Radix UI 기반 컴포넌트
- 인증: 토큰 기반(로컬스토리지), Auth Context
- API: REST (Mavis Backend), 현재 Auth/User 중심 스펙

## 9. 데이터 및 상태 (현황)

- 피드/아이디어/로드맵/팀 데이터는 Mock 기반으로 구성
- 실제 API 연동은 인증/사용자 중심으로 제한적

## 10. 제약 사항 및 리스크

- AI 분석 결과는 백엔드 의존성이 높음 (실시간 분석 품질/지연 리스크)
- 아이디어/피드/팀/로드맵 API 미구현 상태일 가능성
- 해커톤 성격 상 기능 범위/일정 제약

## 11. 범위 제외 (Non-Goals)

- 결제/수익화 모델
- 고도화된 팀 매칭 알고리즘
- 상세 KPI 대시보드

## 12. 오픈 이슈

- AI 분석 API의 실제 동작 및 응답 스키마 확정 필요
- 아이디어/피드/팀/로드맵에 대한 백엔드 API 설계/구현 여부 확인
- 커뮤니티 관리(신고/모더레이션) 범위 결정

## 13. 최근 변경 사항 (홈페이지 카피/구성)

- 메인 히어로 메시지를 Micro Business/실 사용 서비스/경력 중심으로 재구성
- 타이틀에 타이핑 효과 적용 및 레이아웃 흔들림 방지
- "왜 지금, 나만의 아이디어로 시작해야 할까요?" 섹션 추가 + 유튜브 임베드
- 기능 섹션 메시지를 아이디어 → 매출 전환 흐름으로 수정
- 마지막 CTA에 "취업 시장에서 원하는 경력" 강조 문구 반영
- CJK 줄바꿈 개선을 위해 전역 keep-all 적용
- 비로그인 상태에서 AI 분석/로드맵 진입 및 댓글 작성 시 알림 + 로그인 모달 노출
- AI 분석/로드맵 메뉴 및 주요 진입 링크에서 로그인 전 이동 차단 처리
